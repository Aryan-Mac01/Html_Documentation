HTML Documentation

1. What is HTML and its Role in Web Development?

HTML (HyperText Markup Language) is the foundational language used to create and structure content on the web. It provides the building blocks for webpages by allowing developers to define the structure and layout of text, images, links, and more.

Features of HTML:

  • Semantic Structure
  • Cross-Platform
  • Hypertext
  • Media Embedding

2. HTML Document Structure and Essential Elements

An HTML document follows a standard structure, beginning with a <!DOCTYPE> declaration and including several core elements.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Sample HTML Document</title>
</head>
<body>
    <h1>Welcome to HTML Basics</h1>
    <p>This is a sample paragraph.</p>
    <a href="https://www.example.com">Visit Example</a>
    <img src="image.jpg" alt="Sample Image">
</body>
</html>

3. HTML Elements

HTML elements are individual components that define parts of a webpage, each typically having an opening and closing tag.

<p>This is a paragraph.</p>

4. HTML Attributes

Attributes provide additional information about elements, often used to specify element properties.

<a href="https://www.example.com" target="_blank">Example Link</a>

5. HTML Headings

Headings are created using <h1> to <h6> tags, where <h1> is the most important and <h6> the least.

<h1>Main Heading</h1>
<h2>Sub Heading</h2>

6. HTML Paragraphs

Paragraphs in HTML are created with the <p> tag, used to wrap blocks of text.

<p>This is a paragraph of text.</p>

7. HTML Styles

Inline styles can be applied to HTML elements using the style attribute, commonly used for quick, one-off changes.

<p style="color:blue;">This text is blue.</p>

8. HTML Formatting

HTML offers various tags for text formatting, such as <b> for bold, <i> for italics, and <u> for underline.

<b>Bold Text</b>

9. HTML Quotations

Quotations in HTML are created with the <blockquote> and <q> tags.

<blockquote>This is a blockquote.</blockquote>

10. HTML Comments

HTML comments are added using the <!-- --> syntax. These comments are not visible in the rendered webpage.

<!-- This is a comment -->

11. HTML Colors

Colors can be applied to HTML elements using the style attribute or CSS. Colors can be specified using color names, hex codes, RGB values, etc.

<p style="color:red;">This text is red.</p>

12. HTML CSS

CSS can be used to style HTML elements, either inline, internal, or externally linked.

<style>
    p {
        color: blue;
    }
</style>

13. HTML Links

Links are created using the <a> tag. Links can point to internal pages or external websites.

<a href="https://www.example.com">Visit Example</a>

14. HTML Images

Images are inserted using the <img> tag. The src attribute defines the image source.

<img src="image.jpg" alt="Sample Image">

15. HTML Favicon

Favicon is the small image that appears next to the page title in the browser tab, added using the <link> tag in the head section.

<link rel="icon" href="favicon.ico">

16. HTML Page Title

The page title is defined using the <title> tag in the head section.

<title>Sample Page</title>

17. HTML Tables

Tables are defined with <table> and rows with <tr>. Data is added inside <td> tags.

<table>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

18. HTML Lists

HTML supports ordered (<ol>) and unordered (<ul>) lists.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

19. HTML Block & Inline

Elements can either be block-level or inline. Block-level elements take up the full width, while inline elements only take up as much space as their content.

20. HTML Div

The <div> element is used to group sections of content.

<div>This is a div container</div>

21. HTML Classes

Classes are used to group elements and apply shared styles. The class attribute is added to the element.

<p class="myClass">Styled paragraph</p>

22. HTML Id

Ids are used to identify specific elements. An id should be unique within a page.

<div id="unique">Unique element</div>

23. HTML Iframes

The <iframe> tag is used to embed another HTML page within the current page.

<iframe src="page.html"></iframe>

24. HTML JavaScript

JavaScript is added to HTML using the <script> tag, either inline or externally linked.

<script>alert("Hello!");</script>

25. HTML File Paths

File paths are used to link to external files like images, stylesheets, and scripts. These can be relative or absolute paths.

26. HTML Head

The <head> section contains metadata about the document, including the title, link to external stylesheets, and scripts.

27. HTML Layout

HTML layout is achieved using a combination of block-level elements like <header>, <main>, and <footer>.

28. HTML Responsive

Responsive design can be achieved using CSS media queries to adjust the layout for different screen sizes.

29. HTML Computer Code

Computer code is typically displayed using the <code> and <pre> tags for proper formatting.

console.log("Hello, World!");

30. HTML Semantics

HTML semantics refers to the use of HTML tags that clearly describe their meaning in a human- and machine-readable way.

31. HTML Style Guide

HTML style guides provide best practices on writing clean, readable, and consistent code.

32. HTML Entities

Entities are special characters represented by their corresponding HTML code, such as & for an ampersand.

33. HTML Symbols

HTML symbols can be added using their respective entity names or codes.

34. HTML Emojis

Emojis can be inserted directly into the HTML document as Unicode characters or through their corresponding entity.

35. HTML Charsets

Charsets are defined using the <meta charset="UTF-8"> tag to specify how the document's characters are encoded.

36. HTML URL Encode

URL encoding replaces special characters in URLs with a "%" followed by two hexadecimal digits.

37. HTML vs. XHTML

XHTML is a stricter version of HTML that follows XML rules. It requires all elements to be properly closed and nested.
